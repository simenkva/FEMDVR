# FEMDVR

A Python implementation of Finite Element Method with Discrete Variable Representation for quantum mechanical calculations.

## Contents

- `femdvr.py` - Main FEM-DVR class for constructing differentiation matrices
- `lobatto.py` - Gauss-Lobatto quadrature base classes  
- `hydrogen_convergence.ipynb` - Example: hydrogen atom eigenvalue problem

## Usage

Combines finite element flexibility with spectral method accuracy for solving quantum eigenvalue problems on complex grids.
